🎯 Objetivo

Implementar un selector de capas base (calles, satélite, terreno), controlar la cámara con animaciones, y crear marcadores que muestren información al tocarlos.

🗺️ Múltiples Capas Base

En QGIS cambias entre mapas base con un clic. En Flutter, cambias la URL de los tiles.

Proveedores de Tiles Gratuitos

Cada proveedor genera los tiles (imágenes cuadradas del mapa) con un estilo diferente. Definimos las URLs como constantes para reutilizarlas:

Dart
class TileProviders {
  static const String osm =
    'https://tile.openstreetmap.org/{z}/{x}/{y}.png';

  static const String satelite =
    'https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}';

  static const String terreno =
    'https://server.arcgisonline.com/ArcGIS/rest/services/World_Topo_Map/MapServer/tile/{z}/{y}/{x}';

  static const String oscuro =
    'https://basemaps.cartocdn.com/dark_all/{z}/{x}/{y}@2x.png';
}
Es como tener varios mapas base en QGIS (OSM, Google, Bing) y cambiar entre ellos desde el panel de capas.
Selector con DropdownButton

Necesitamos una variable de estado para controlar qué capa está activa, y un widget para que el usuario elija:

Dart
// Variables de estado
String _urlCapaActiva = TileProviders.osm;
String _nombreCapa = 'Calles';

final Map<String, String> _capas = {
  'Calles':    TileProviders.osm,
  'Satélite': TileProviders.satelite,
  'Terreno':  TileProviders.terreno,
  'Oscuro':   TileProviders.oscuro,
};

void _cambiarCapa(String nombre, String url) {
  setState(() {
    _nombreCapa = nombre;
    _urlCapaActiva = url;
  });
}
⚠️
Si olvidas el setState(), la variable cambia pero Flutter no redibuja el mapa. Siempre que modifiques el estado visual, envuélvelo en setState().
TileLayer dinámico

Ahora el TileLayer usa la variable en vez de una URL fija:

Dart
TileLayer(
  urlTemplate: _urlCapaActiva,  // ← Dinámico
  userAgentPackageName: 'com.example.geo_collect',
)

Animaciones suaves

Para que el mapa se mueva suavemente (como Google Maps), necesitamos animaciones. Primero, agrega TickerProviderStateMixin al State:

Dart
class _MapaScreenState extends State<MapaScreen>
    with TickerProviderStateMixin {

  void _animarHacia(LatLng destino, double zoomFinal) {
    final latTween = Tween<double>(
      begin: _mapController.camera.center.latitude,
      end: destino.latitude,
    );
    final lngTween = Tween<double>(
      begin: _mapController.camera.center.longitude,
      end: destino.longitude,
    );
    final zoomTween = Tween<double>(
      begin: _mapController.camera.zoom,
      end: zoomFinal,
    );

    final ctrl = AnimationController(
      duration: const Duration(milliseconds: 800),
      vsync: this,
    );

    final curve = CurvedAnimation(
      parent: ctrl, curve: Curves.easeInOut,
    );

    ctrl.addListener(() {
      _mapController.move(
        LatLng(latTween.evaluate(curve),
              lngTween.evaluate(curve)),
        zoomTween.evaluate(curve),
      );
    });

    ctrl.addStatusListener((s) {
      if (s == AnimationStatus.completed) ctrl.dispose();
    });

    ctrl.forward();
  }
}
💡
No necesitas memorizar este código. Es un patrón que copias y reutilizas. Lo importante es saber que existe y cuándo usarlo.

💬 Marcadores con Popups

Instalar flutter_map_marker_popup
yaml — pubspec.yaml
dependencies:
  flutter_map_marker_popup: ^7.0.0
Terminal
flutter pub get
Reemplazar MarkerLayer con PopupMarkerLayer

En vez de MarkerLayer, usamos PopupMarkerLayer que agrega el popup automáticamente al tocar:

Dart
import 'package:flutter_map_marker_popup/flutter_map_marker_popup.dart';

final PopupController _popupCtrl = PopupController();

// Dentro de FlutterMap children:
PopupMarkerLayer(
  options: PopupMarkerLayerOptions(
    popupController: _popupCtrl,
    markers: misMarcadores,
    popupDisplayOptions: PopupDisplayOptions(
      builder: (BuildContext ctx, Marker marker) {
        return _miPopup(marker);
      },
    ),
  ),
),
Es como hacer clic en un feature de QGIS y ver sus atributos en un panel. Aquí tocas el marcador y aparece la info.
Diseñar el Widget del Popup
Dart
Widget _miPopup(Map<String, dynamic> punto) {
  return Container(
    width: 200,
    padding: const EdgeInsets.all(12),
    decoration: BoxDecoration(
      color: Colors.white,
      borderRadius: BorderRadius.circular(10),
      boxShadow: [BoxShadow(
        color: Colors.black26, blurRadius: 8,
      )],
    ),
    child: Column(
      mainAxisSize: MainAxisSize.min,
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        Text(punto['nombre'],
          style: const TextStyle(fontWeight: FontWeight.bold)),
        Text('Tipo: ${punto['tipo']}',
          style: TextStyle(fontSize: 12, color: Colors.grey)),
      ],
    ),
  );
}

📏 Polilíneas — Rutas y Caminos

Una polilínea conecta puntos con líneas. En GIS: rutas, ríos, carreteras.

Dibujar una ruta
Dart
// Puntos de una ruta real (busca los tuyos en Google Maps)
final List<LatLng> _miRuta = [
  const LatLng(-17.3935, -66.1570),  // Punto A
  const LatLng(-17.3940, -66.1530),  // Punto B
  const LatLng(-17.3960, -66.1510),  // Punto C
];

// Dentro de FlutterMap children:
PolylineLayer(
  polylines: [
    Polyline(
      points: _miRuta,
      color: Colors.blue,
      strokeWidth: 4.0,
    ),
  ],
),
Digitalizar una línea en QGIS: clic, clic, clic → se conectan los puntos. Aquí defines las coordenadas y Flutter dibuja la línea.

⬡ Polígonos — Áreas y Zonas

Dibujar una zona de estudio
Dart
final List<LatLng> _zona = [
  const LatLng(-17.390, -66.160),
  const LatLng(-17.390, -66.148),
  const LatLng(-17.400, -66.148),
  const LatLng(-17.400, -66.160),
];

PolygonLayer(
  polygons: [
    Polygon(
      points: _zona,
      color: Colors.teal.withOpacity(0.2),
      borderColor: Colors.teal,
      borderStrokeWidth: 2.0,
      isFilled: true,
    ),
  ],
),
💡
Usa withOpacity(0.2) para que el relleno sea semi-transparente y puedas ver el mapa base debajo — igual que la transparencia de capas en QGIS.

👆 Eventos de Interacción

onTap, onLongPress, onPositionChanged
Dart
MapOptions(
  initialCenter: _centro,
  initialZoom: 14,

  // Toque simple → mostrar coordenadas
  onTap: (tapPos, latLng) {
    print('Tap en: $latLng');
  },

  // Toque largo → agregar marcador
  onLongPress: (tapPos, latLng) {
    _agregarPunto(latLng);
  },

  // Al mover el mapa → actualizar barra
  onPositionChanged: (camera, hasGesture) {
    setState(() {
      _zoomActual = camera.zoom;
      _centroActual = camera.center;
    });
  },
),
Agregar marcadores donde toca el usuario
Dart
List<LatLng> _misMarkers = [];

void _agregarPunto(LatLng pos) {
  setState(() {
    _misMarkers.add(pos);
  });
}

// En FlutterMap children:
MarkerLayer(
  markers: _misMarkers.map((p) {
    return Marker(
      point: p, width: 30, height: 30,
      child: const Icon(Icons.push_pin,
        color: Colors.purple, size: 28),
    );
  }).toList(),
),
Esto es captura de puntos en campo: el usuario toca dónde quiere registrar un punto y se agrega al mapa. En el Módulo 5 le agregaremos formulario de atributos.

📖 Glosario — Clase 04

PolylineSecuencia de puntos conectados por segmentos de línea. Representa rutas, ríos, carreteras.
PolygonÁrea cerrada definida por vértices. Representa municipios, parcelas, zonas de estudio.
onTapEvento que se dispara al tocar brevemente el mapa. Recibe la posición en LatLng.
onLongPressEvento al mantener presionado el mapa. Ideal para agregar puntos.
onPositionChangedSe ejecuta cada vez que el mapa se mueve o hace zoom. Útil para actualizar info en tiempo real.
withOpacity()Método que aplica transparencia a un color. 0.0 = invisible, 1.0 = sólido.
SnackBarBarra de mensaje temporal que aparece abajo de la pantalla.

📄 ¿Qué es GeoJSON?

El formato universal de datos geográficos

GeoJSON es un formato de texto para representar datos geográficos. Es el formato más universal de intercambio en GIS moderno: lo leen QGIS, ArcGIS, PostGIS, Google Maps, Leaflet y cualquier herramienta actual.

Un archivo GeoJSON es simplemente texto JSON organizado con una estructura específica:

GeoJSON
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [-68.1336, -16.4955]
      },
      "properties": {
        "nombre": "Poste eléctrico #42",
        "tipo": "Poste",
        "estado": "Bueno"
      }
    }
  ]
}
FeatureCollection
La colección completa = una capa en QGIS
geometry
DÓNDE está el punto (coordenadas)
properties
QUÉ es el punto (atributos)
⚠️
¡CUIDADO con el orden de coordenadas! En GeoJSON las coordenadas van [longitud, latitud] — al revés de como las usamos en Flutter (LatLng(lat, lng)). Si tus puntos aparecen en el océano, es por esto.
Un FeatureCollection de GeoJSON es exactamente como una capa vectorial de QGIS: una colección de geometrías, cada una con sus atributos en la tabla.

⚙️ Generar GeoJSON en Dart

Función que convierte tus puntos a GeoJSON
Dart
String generarGeoJSON(List<Map<String, dynamic>> puntos) {
  final features = puntos.map((p) {
    return {
      'type': 'Feature',
      'geometry': {
        'type': 'Point',
        'coordinates': [
          p['longitud'],  // ← lng primero
          p['latitud'],   // ← lat segundo
        ],
      },
      'properties': {
        'nombre': p['nombre'],
        'tipo': p['tipo'],
        'estado': p['estado'],
        'fecha': p['fecha'],
      },
    };
  }).toList();

  final geojson = {
    'type': 'FeatureCollection',
    'features': features,
  };

  return const JsonEncoder.withIndent('  ').convert(geojson);
}

📤 Guardar y Compartir

Nuevas dependencias

dependencies:
  path_provider: ^2.1.0    # ← Acceso a carpetas del teléfono

💡
path_provider te da la ruta de las carpetas del teléfono (documentos, caché, etc.). share_plus abre el menú de compartir de Android (WhatsApp, email, Drive, etc.).
Guardar archivo y compartir
Dart
import 'dart:io';
import 'package:path_provider/path_provider.dart';
import 'package:share_plus/share_plus.dart';

Future<void> _exportar() async {
  // 1. Generar el texto GeoJSON
  final texto = generarGeoJSON(_puntos);

  // 2. Obtener carpeta de documentos
  final dir = await getApplicationDocumentsDirectory();

  // 3. Crear archivo con nombre único
  final ts = DateTime.now().millisecondsSinceEpoch;
  final archivo = File('${dir.path}/geocollect_$ts.geojson');
  await archivo.writeAsString(texto);

  // 4. Abrir menú de compartir
  await Share.shareXFiles(
    [XFile(archivo.path)],
    text: 'GeoCollect — ${_puntos.length} puntos',
  );
}

📦 Generar APK

¿Qué es un APK?

APK = Android Package. Es el archivo que se instala en un teléfono Android. Es como un .exe en Windows o un .dmg en Mac. Cuando generas el APK, Flutter compila todo tu código Dart a código nativo optimizado, incluye los recursos, y empaqueta todo en un solo archivo.

Terminal
flutter build apk --release

El APK se genera en:

Ruta
build/app/outputs/flutter-apk/app-release.apk
💡
El build tarda 2-3 minutos. La versión release es más pequeña y rápida que la versión debug que usamos en desarrollo.
Instalar en otro teléfono

Para instalar tu APK en otro teléfono Android:

1. Envía el archivo APK por WhatsApp, Google Drive o cable USB
2. En el teléfono receptor: Configuración → Seguridad → "Fuentes desconocidas" → Activar
3. Abre el archivo APK y presiona "Instalar"
4. ¡La app está lista para usar!

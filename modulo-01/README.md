# 📱 Flutter GIS — Desarrollo de Apps Geoespaciales con Flutter

> Curso práctico de 30 horas para construir una app Android real con mapas interactivos, GPS, capas WMS de GeoServer y exportación a GeoJSON.

## 🎯 ¿Qué vas a construir?

Una app Android completa llamada **GeoCollect** que:

- Muestra mapas interactivos con múltiples capas base
- Se conecta a un servidor GeoServer real y visualiza capas WMS profesionales
- Captura coordenadas GPS en campo con formularios de atributos
- Visualiza tus datos propios sobre las capas del servidor
- Exporta datos a GeoJSON compatible con QGIS
- Se instala como APK en cualquier teléfono Android

- Conexión a internet estable
